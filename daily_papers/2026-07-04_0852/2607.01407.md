# Vitriflow: calibrated amorphous structure ensembles from melt-quench simulation

**arXiv**: [2607.01407](https://arxiv.org/abs/2607.01407)
**Authors**: Jonathon Cottom, Robin Delhomme, Emilia Olsson
**Categories**: cond-mat.mtrl-sci

## Abstract

arXiv:2607.01407v1 Announce Type: new 
Abstract: Melt--quench molecular dynamics is widely used to construct amorphous materials models, but the resulting ensemble is defined by choices that are often made implicitly: numerical settings, melt temperature, liquid-hold time, quench rate, system size, and post-generation screening. We introduce vitriflow, a computational materials methodology that turns these choices into an explicit decision chain. The framework couples numerical stability, descriptor-based protocol calibration, user-defined artefact screening, and statistical convergence of the generated analysis ensemble in a material-specific descriptor space. We demonstrate the approach for a-SiO$_2$, a-Si$_3$N$_4$, and a-Sm$_2$O$_3$, which respectively test tetrahedral network fidelity, MG2 $\rightarrow$ PBE $\rightarrow$ HSE06 DFT refinement of a heteropolar nitride, and amorphous/crystal discrimination in a mixed-coordination rare-earth oxide. vitriflow separates defect-free from oxygen-bridge-defective silica, quantifies DFT-refinement response in a common a-Si$_3$N$_4$ structural population, and removes recrystallised Sm$_2$O$_3$ structures without imposing fixed coordination. The result is a reproducible route for generating amorphous ensembles whose numerical settings, thermal protocol, screening actions, and statistical precision are selected from the materials question rather than assumed a priori.

### Vitriflow: calibrated amorphous structure ensembles from melt-quench simulation - 详细分析笔记

- **一句话总结**: 提出Vitriflow校准框架，将熔融淬火模拟从任意过程变为可重复、可验证的非晶结构生成决策流程。

- **研究背景与动机**:
    - **解决的科学问题**: **熔融淬火分子动力学**（Melt-quench MD）是构建非晶材料模型的常用方法。然而，其结果高度依赖于大量隐含的、未经验证的经验参数（如熔融温度、淬火速率、系统尺寸等），导致生成的结构集缺乏**统计可重复性**和**结构可靠性**。论文旨在解决非晶模拟领域“执行可重复”但“结构不可信”的深层问题。
    - **与已有工作的差异**: 现有工作流框架（AiiDA、FireWorks）保证了“执行的可重复性”，但未解决“结构的可重复性”。Vitriflow首次将**数值稳定性验证**、**热力学协议校准**、**用户定义伪影筛选**和**统计收敛性**集成为一个材料特定的决策链，确保生成的非晶结构集物理合理，且置信度可控。

- **核心材料/体系**:
    - **体系**: 论文验证了三个不同的无机非晶材料体系，以展示框架的通用性：
        1.  **非晶二氧化硅 (a-SiO₂)**: 使用 **SHIK-1** 势函数，测试对**四面体网络**保真度的控制。
        2.  **非晶氮化硅 (a-Si₃N₄)**: 使用 **MG2** 势函数生成初始结构，随后使用 **PBE** 和 **HSE06** 密度泛函理论（DFT）进行弛豫，测试**多保真度精炼**下的结构响应。
        3.  **非晶氧化钐 (a-Sm₂O₃)**: 使用 **库仑-白金汉势**，测试在**混合配位**稀土氧化物中区分非晶与再结晶结构的能力。
    - **制备方法**: 经典的**熔融淬火分子动力学**模拟，但关键参数（熔融温度、保温时间、冷却速率、系统尺寸）由Vitriflow框架通过系统性校准确定。

- **关键物理机理**:
    - **路径依赖性**: 熔融淬火得到的非晶状态高度依赖于其历史路径（熔融温度、保温时间、冷却速率等）。不同路径会导致最终结构在**密度、配位数、键角分布、环统计**和**中程有序**（如第一尖锐衍射峰）方面存在差异。
    - **决策链机制**: Vitriflow框架的核心是通过一个材料特定的决策序列，确保模拟的物理合理性。该序列包括（具体算法见下文“主要表征结果”中的方法细节）：
        1.  **数值稳定性验证**: 通过能量漂移、压力控制误差等指标，筛选出数值稳定的模拟设置。
        2.  **热力学协议校准**: 通过分析熔体扩散、结构序参数随温度的变化，确定**液态起始温度** $T_{\mathrm{liq}}$，进而设定合理的**熔融温度** ($T_{\mathrm{high}}$)，并通过分析均方位移确定**最短保温时间** ($t^{\star}_{\mathrm{melt}}$)。
        3.  **冷却速率与尺寸选择**: 通过**描述符等价性**概念，选择在统计上产生与慢冷却/大系统不可区分的结构的最快冷却速率/最小系统尺寸，以节省计算成本。
        4.  **伪影筛选与统计收敛性**: 在生成的结构系综中，基于用户定义的描述符（如配位数规则、结晶度指标）进行筛选（排除或标记），并判断剩余“分析系综”是否在描述符空间内达到统计收敛。

- **主要表征结果**:

    **方法细节 (关键公式与算法)**:
    *   **描述符定义**: 框架基于**描述符映射** $Φ(x; D) = (d_1(x), d_2(x), …, d_p(x))$ 驱动。$D$ 是材料特定的描述符集（如密度、RDF、配位数、环统计）。
    *   **数值稳定性筛选**: 通过定义四个误差指标（温度控制误差 $\epsilon_T$、压力控制误差 $\epsilon_P$、体积合理性因子 $R_V$ 和无失败）来界定**数值允许设置集合** $\Theta_{\text{adm}}$。性能分数 $S_{\text{pf}}$ 用于在允许集合内选择最优设置。
    *   **热力学协议校准**:
        *   **液态起始点确定**: 使用扩散系数$D$、第一RDF峰值高度$H$和宽度$W$作为描述符，通过其随温度变化的**标准化导数** $\Sigma(T_i)$ 和结构门控函数 $L_i$ (需要满足扩散距离 $\chi_i \geq \chi_*$，峰值高度 $H_i \leq \eta_H H_0$，宽度 $W_i \geq \eta_W W_0$) 来识别熔体进入液相区的温度 $T_{\mathrm{liq}}$。
        *   **液态保温时间**: 通过独立副本的均方位移达到**α倍**平均键长的时间 ($t_{\text{dis},k}$ ) 来确定，取最坏情况 $\max_k$。同时要求密度和能量达到**统计平稳** ($\delta_x$ 小于阈值)。
        *   **冷却速率与尺寸选择**: 核心思想是**描述符等价性**。对于候选冷却速率 $\dot{T}_i$，计算其与最慢参考速率在描述符$j$上的 “**等价性指标**” $E_{ij}$。若 $E_{ij}$ 小于用户指定的 **描述符容差** $\tau_j(\mu_{\text{ref}})$，则认为其等价。最终选择满足所有必需描述符等价性的**最快**冷却速率和**最小**系统尺寸。
    *   **伪影筛选与分析系综收敛**:
        *   **筛选**: 对每个生成的结构 $x_b$，应用用户定义的**筛选规则** $\Psi_s$，生成标签 $A_{bs}=1$ (通过) 或 $0$ (失败)。通过 `exclude` 设置来控制失败标签的结构是**排除**还是仅**标记**。分析系综 $\mathcal{E}_{\text{ana}}$ 由所有最终保留的结构组成。
        *   **收敛性度量**: 对于分析系综中的描述符$j$，计算其**置信半宽** $h_j(n)$ 与**容差** $\tau_j(\hat{\mu}_j(n))$ 的比值，即**置信宽度比** $R_j(n)$。当系综包含的所有收敛描述符的 $Q(n) = \max_{j \in \mathcal{D}_{\text{conv}}} R_j(n) \leq q$ (例如 $q=0.2$) 时，认为该系综是收敛的。

    **关键结果**:
    1.  **a-SiO₂ 体系**: **精确配位筛选** (Si:O=4, O:Si=2) 成功区分了“无缺陷” (1872个) 和“**氧桥缺陷**”(661个) 两个不同的非晶结构群体。缺陷结构主要是氧位配位错误，其密度略高 ($2.233 \pm 0.002$ vs $2.217 \pm 0.001$ g/cm³)，且Si-O第一近邻峰位有微小偏移。该实验证明，不经过筛选的“平均”结构会混合两种不同的物理状态。
    2.  **a-Si₃N₄ 体系**: **多保真度精炼层次**清晰显示了从MG2经PBE到HSE06的结构演变。虽然第一壳层的Si-N键长相对稳健，但**密度、残余缺陷分数和中程拓扑**（如环统计）对DFT水平的响应非常敏感。例如，PBE导致密度相比MG2下降了2.48%，而HSE06的密度与MG2非常接近，仅下降0.22%。环的平均尺寸从MG2的6.316增加到PBE/HSE06的6.658，变化显著。
    3.  **a-Sm₂O₃ 体系**: **晶体序参量筛选** 成功去除了熔融淬火过程中发生**再结晶**的结构，而无需预设固定的配位数规则。430个保留的“非晶”结构与80个“类晶”结构在**结晶分数**、**局部序参数** $\bar q_6$ 和衍射特征上存在显著差异。保留的非晶结构呈现混合配位特征（Sm-O主要为五、六配位；O-Sm主要为四配位），符合预期。

- **性能指标**: 本文关注结构描述的统计精度，而非材料性能。
    - **系综收敛标准**: 默认目标置信宽度比 $q_{\text{conv}} = 0.2$。
    - **描述符容差**: 用户可设定绝对容差 $\tau_{\text{abs}}$ 和相对容差 $\tau_{\text{rel}}$，如 $\tau_j(\mu) = \max(\tau_{\text{abs}}, \tau_{\text{rel}}|\mu|)$。
    - **筛选产出率**: a-SiO₂: 73.9% (1872/2533) 通过筛选；a-Sm₂O₃: 84.3% (430/510) 通过筛选。

- **创新点**:
    1.  **从“执行可重复”到“结构可重复”**: 核心创新在于提出了一个系统性校准框架，将工作流重点从“如何运行模拟”提升到“如何生成有物理意义的非晶结构”。
    2.  **材料特定的描述符决策空间**: 框架是**描述符驱动**的，以材料特定的描述符（如配位数、环统计、局部序参数）作为校准和筛选的标准，比单纯依赖密度或能量等全局标量更具物理洞察力。
    3.  **结构伪影的显式处理**: 通过用户定义的、带 `exclude` 选项的筛选规则，框架能够显式地识别并排除或标记特定结构缺陷（如氧桥、再结晶），这是以往工作流中缺失的。

- **与我研究的相关性**:
    - **方法学高度匹配**: 我的知识库主要由高分子物理研究组成，特别是关于**高分子玻璃化转变**(Tg)、**熔体动力学**(Rouse/Reptation模型) 和**受限效应**。Vitriflow提出的**熔融淬火校准方法论**，其核心思想（消除历史记忆、冷却速率影响、系统尺寸效应）与高分子物理中研究玻璃态结构时所面临的挑战**完全一致**。虽然对象是无机非晶，但其解决问题的**分析逻辑**（如通过扩散系数判断熔体平衡，通过环统计分析拓扑）可**直接移植**到高分子体系的研究中。例如，高分子体系中的**缠结**、**链构象**（如回转半径）等可被定义为描述符。其**描述符等价性**概念，可以有效地指导我们在选择模拟高分子的冷却速率或链长时，做出有物理依据的判断，而非随意选择。

- **局限性**:
    - **计算成本**: 要进行全面的参数校准（扫描不同熔融温度、冷却速率、尺寸、独立副本）需要大量的模拟资源，可能限制在超大规模高通量筛选中的应用。
    - **势函数依赖性**: 整个框架的有效性高度依赖于所选取的**经验势**或**机器学习势**的精度。如果势函数本身不能正确描述某种关键结构（如a-SiO₂中的氧桥缺陷），校准也无法弥补其缺陷。该论文承认其不替代良好的力场。
    - **非平衡态特性**: 熔融淬火对于非玻璃形成体系而言，是一种“受控的路径”而非真实的“合成过程”。该方法更多是生成物理上合理的结构模型，而非精确重现实验制备过程。对于本身无法通过熔融淬火法制备的高分子体系，该方法可能不适用。

- **后续可跟进方向**:
    - **方法的改进**: 如何将**机器学习势函数**的UQ（不确定性量化）集成到校准流程中？能否实现校准过程的自动化，减少人为设定的阈值？
    - **与实验对比**: 如何将Vitriflow生成的系综结构信息，与实验（如X射线/中子散射、固态NMR）进行更直接的比对，以验证其“结构可重复性”是否对应于“实验可重现性”？
    - **高分子体系的直接应用开发**: 开发一个针对高分子的Vitriflow分支，定义高分子的特定描述符集（如**链缠结密度**、**回转半径**、**末端距分布**、**链内环分布**），并研究其在**聚合物共混物**、**嵌段共聚物**和**纳米复合材料受限**等复杂体系中的应用。

    **回答第一轮提出的问题**:
    1.  **扩散系数阈值/判定标准**: 论文定义了**扩散冻结温度**$T_f$，即温度序列表中扩散系数$D(T_j) \leq D^\ast$ 的第一个温度。文中未明确给出 $D^\ast$ 的具体数值，但其包含在用于识别液态的**结构门控函数** $L_i$ 中，要求 $\chi_i \geq \chi_\star$，其中 $\chi_i = \sqrt{\mathrm{MSD}_i} / \ell_i$ (均方位移与平均原子间距之比的平方根)。因此，判断熔体平衡的依据本质上是一个**与原子尺寸相关的扩散距离指标**，而非固定的扩散系数值。
    2.  **环统计及系统尺寸**: 论文对a-SiO₂使用 **投影态Si环统计** (projected Si-ring statistics)，对a-Si₃N₄使用 **Franzblau键图环算法**。环尺寸分布随冷却速率和系统尺寸的变化关系在正文中未给出具体图表，而是引用到**补充材料 Figs. S1 和 S2**。但也给出了a-Si₃N₄中等效环尺寸的量化结果：MG2 (6.316) $\rightarrow$ PBE (6.658) $\rightarrow$ HSE06 (6.656)，显示DFT精炼会显著增加环的平均尺寸。文中建议的最小系统尺寸标准同样是基于**描述符等价性**，即选择与最大测试系统在描述符上不可区分的最小系统。
    3.  **高分子链拓扑描述符**: 论文本身未处理此问题。但这一思路正是其价值所在。对于高分子，**缠结**可以被量化为**缠结分子量** ($M_e$) 或**每个链的缠结点数**。**支化**可以用**支化度**、**回旋半径比** $(R_g^2/R_{g, \text{linear}}^2)$ 等描述符量化。这些都可以作为 **结构伪影** 筛选的标准（例如，排除过度缠结或未缠结的体系）或 **协议校准** 的描述符。对算法的主要修改在于定义这些描述符的计算方法，并可能需要引入基于拓扑的网络分析（如Knot理论和Primitive Path Analysis）。
    4.  **伪影筛选标准**:
        *   **a-SiO₂ 的氧桥缺陷**: 筛选标准是 **第一壳层配位数**：精确的Si:O = 4 和 O:Si = 2。任何不满足该条件的结构都被标记为含氧桥缺陷。这是一个基于**化学计量的硬几何规则**。
        *   **高分子体系的常见伪影**: 对于高分子熔融淬火，常见的伪影可能包括：
            1.  **链间贯穿/缠结不足**: 在极快淬火下，链来不及形成足够的物理缠结，导致模拟的玻璃态力学性能偏离实验。
            2.  **反常的环化**: 在熔融状态下，链末端的环化反应（如果势函数允许）可能导致非物理的小环结构。
            3.  **构象冻结**: 构象空间采样不足，导致大量链被困在单一的高能构象中（如全部为全同构象）。
            4.  **未溶链条**: 在多组分体系中，某些组分未能充分混合，形成尺寸大于统计预期的团簇。这些都可以通过定义**描述符**（如链的$R_g$分布、芳香环的二面角分布、团簇尺寸分布）并设定**阈值**来进行伪影筛选。
