# Polymer Physics / 高分子物理

本分类涵盖与**高分子物理与物理化学**相关的论文，包括但不限于：

- 高分子链构象与动力学（Rouse/Zimm 模型、爬行管模型）
- 高分子结晶与熔融行为（成核、球晶生长、结晶动力学）
- 玻璃化转变与弛豫行为（α/β弛豫、WLF/VFT 方程）
- 微相分离与自组装（嵌段共聚物、胶束、囊泡）
- 高分子流变学与粘弹性（线性/非线性粘弹性、时间-温度叠加）
- 高分子网络与凝胶（化学/物理交联、溶胀、网络拓扑）
- 高分子共混物与复合材料（相容性、界面、增强机理）
- 高分子电解质与离子传导（离子传输机制、离子液体、PEO基电解质）
- 水凝胶与离子凝胶（韧性、自修复、刺激响应）
- 形状记忆高分子（SMP）
- 压电/摩擦电高分子（PVDF及其共聚物、极化、能量收集）
- 自修复材料（动态共价键、超分子作用、氢键网络）
- 软物质力学（韧性、断裂、疲劳、非线性力学行为）
- 高分子表征方法（DSC, SAXS/WAXS, NMR, AFM, rheology, DMA, TGA, XRD）
- 分子动力学模拟（粗粒化模型、全原子模拟、DPD、耗散粒子动力学）
- 高分子功能材料（导电高分子、介电弹性体、人工肌肉、电子皮肤）
- 生物高分子与仿生材料（胶原蛋白、纤维素、蛋白质基材料）

**典型关键词**: polymer physics, glass transition, crystallization, viscoelasticity,
rheology, microphase separation, block copolymer, hydrogel, ionogel, self-healing,
piezoelectric, PVDF, elastomer, entanglement, reptation, DSC, SAXS, molecular dynamics,
polyelectrolyte, ionic liquid, shape memory, soft robotics, toughness, fatigue

**典型 arXiv 类别**: cond-mat.soft, cond-mat.mtrl-sci, physics.chem-ph, physics.app-ph
